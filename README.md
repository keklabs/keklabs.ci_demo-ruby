# Jenkins Pipeline Demo

This project is used for demonstration of building Ruby projects with CI-Tools like:

- Jenkins 2.x
- Nexus OSS Repository 3.x
- Ruby MRI 2.x
- Docker 1.12.x
- Jenkins Pipeline 2.x
- Github
- Rake
